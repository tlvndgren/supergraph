# Build a Supergraph

This repository contains the demos for the "Build a Supergraph" series. All you need to follow along is:

- A Github account
- An Apollo GraphOS account (free!)
- An account for either:
  - Google Cloud Platform (GCP) 
  - Amazon Web Services (AWS)
